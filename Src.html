How to Use

To use this code generator, all you have to do is pick a language, enter what you want the AI to code, and pick the creativeness/randomness of the code, and voila! Still confused/need help? Read more below!

Languages

This generator allows you to generate code in five different languages; Python, JS, Java, CSS, and C++. The best languages to use for accurate and correct code are Python and JS, while the others may produce very inaccurate/non-working code.

Good Prompts

In order for the AI to produce the best responses, you want to ensure your prompts include enough detail but not too much detail as to confuse/mislead the AI. Here are some example prompts that work exceptionally with the AI:

  • Write a {language} program that... makes a spooky text based adventure game
  • Write a {language} program that... is a basic calculator and has a GUI using some external library
  • Write a {language} program that... generates random and secure passwords
  • Write a {language} program that... solves quadratic equations including irrational and rational roots
  • And More! The possibilities are literally endless :)

Randomness

Randomness, in technical terms, is the temperature the AI uses when generating responses. More simply, setting a higher temperature makes responses more random/creative, while setting a lower temperature makes responses more direct/straight to the point.
